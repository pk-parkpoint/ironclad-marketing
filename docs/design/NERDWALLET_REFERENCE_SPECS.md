# NerdWallet Reference Specs

These guide templates borrow layout structure, spacing rhythm, and content hierarchy from the provided NerdWallet references.

## Article page reference

- Hero band target height: roughly `354px`
- Desktop content frame: `1152px`
- Desktop article / rail split: `808px / 344px` with a `32px` gap
- Hero H1 size reference: `48px`
- Hero dek reference: `20px`
- Body copy reference: `16px`
- Section heading reference: `24px`
- TOC / utility rail pattern: sticky right column

## Hub page reference

- Section title reference: `36px`
- Tool tile footprint: about `179px x 204px`
- Tool tile title: `16px`
- Tool tile label: `13px`
- Article cards use rounded surfaces and tighter interior padding than the old scaffold

## Font rule

Do not copy the NerdWallet font stack. Keep Ironclad's site font and only copy measurements, spacing, and layout proportions.
